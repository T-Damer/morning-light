## Getting Started

1. Clone the repo: `git clone https://github.com/T-Damer/morning-light.git`
2. Install dependencies: 
```bash
flutter clean
flutter packages get
flutter packages upgrade
```
3. Run the App (in VSCode press F5 if you have flutter extension)
